# Functions

Сигнатура функции
	У каждой функции есть уникальное название и набор параметров, то есть значений, которые мы в неё передаём.

```go
func factorial(n int) int
```

`factorial` — название функции, а `n int` — её параметр, который должен быть целым числом. Как мы и оговаривали ранее, у него может быть любое значение, но обязательно заданного (в нашем случае — целочисленного) типа. Ну и, наконец, `int` — тип возвращаемого значения.

Вот ещё один пример:

```go
func findDiscriminant(a, b, c float64) {
  // Тело функции
}
```

<mark style="background: #FFF3A3A6;">Процедура — подпрограмма, которая только выполняет операции, без возврата значения.</mark>
