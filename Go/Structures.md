# Structures

Объявление структуры выглядит так:

```go
type ИмяСтруктуры struct {
    // переменные (поля) структуры
}
```

Создание экземпляра класса: 

```go
student1 := Student{name: "vasya", age: 15}
```


<mark style="background: #FFF3A3A6;">Методы в Go — это функции, которые связаны с определённой структурой. Они позволяют работать с данными, которые хранятся в структуре, и изменять их состояние.</mark>

Синтаксис метода очень похож на синтаксис функции, но у него есть одно важное отличие: метод должен быть связан с определённой структурой. Для этого перед именем функции указывается имя структуры (так называемый ресивер), с которой она связана:

```go
func (s Student) printData() {
    fmt.Printf("Name: %s, Age: %d\n", s.Name, s.Age)
}
```

Чтобы вызвать метод, мы используем точечную нотацию, такую же, как и для доступа к полям структуры:

```go
student1 := Student{Name: "vasya", Age: 15}
student1.printData()
```


В Go есть два вида ресиверов: с указателем или по значению. Если мы используем ресивер по значению, как в предыдущем примере, объект копируется. Давайте рассмотрим ресивер по указателю:

```go
func (s *Student) printData() {
    fmt.Printf("Name: %s, Age: %d\n", s.Name, s.Age)
}
```
