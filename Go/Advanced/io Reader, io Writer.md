# io Reader, io Writer

## io.Reader

Интерфейс `Reader` в Go определён так:

```go

type Reader interface {
    Read(p []byte) (n int, err error)
}
    
```

![[zamena2 1.png]]
`Read` считывает до `len(p)` байт из источника в заданный байтовый слайс `p` и возвращает количество прочитанных байт и ошибку (если она возникла). Важно проверить возвращаемое значение `n`, так как оно может быть меньше длины `p` , если в источнике меньше данных. Интерфейс Reader используется разными типами пакета `io` (например, `File`, `Buffer`, `Net.Conn`), чтобы читать данные из разных источников.

## Writer

Интерфейс `Writer` в Go определён так:

```go
type Writer interface {
    Write(p []byte) (n int, err error)
}
```

`Write` записывает `len(p)` байт из слайса `p` в место назначения и возвращает количество записанных байт и ошибку (если возникла). Как и в случае с `Reader`, важно проверить возвращаемое значение n , чтобы убедиться, что все данные записаны.

Интерфейс `Writer` также используется разными типами пакета `io` (например, `File`, `Buffer`, `Net.Conn`), чтобы записывать данные в разные места назначения.

`Writer` позволяет не зависеть от того, куда нужно записать данные, в локальный файл или удалённое хранилище. Например, если есть функция, которой нужно передать `io.Writer` для записи:

```go

func WriteData(writer io.Writer, data []byte) error{
    bytesWritten, err := writer.Write(data) // Записываем данные
    if err != nil {
       // TODO: handle error.
    }
    fmt.Printf("Записано %d байт", bytesWritten)
} 
```
