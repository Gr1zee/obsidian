# Numeric types

uint(8 bit to 64) - unsigned int

int(8 bit to 64) - signed int 

float(32 to 64)

byte - представление for uint8
rune - alias for int32

# String types

Представляет собой набор строковых значений. Строковое значение - последовательность байт, возможно пустая.
Длина строки это количество байт, никогда не отрицательная.
Строки невозможно изменять.
