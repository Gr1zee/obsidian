# Strings. Runes

Строка — это набор символов в двойных кавычках. В строках хранят текстовую информацию: имена, адреса и сообщения об ошибках. Например, вот такой код создаёт переменную `name` со строкой `"Андрей"`:

```go
name := "Андрей"
```

<mark style="background: #FFF3A3A6;">А вот и первая не очевидная особенность строки: после создания изменить её уже нельзя.</mark> Зато можно создавать новые строки на основе старых.

Получение Unicode: 
```go
name := "Yandex"
firstLetter := name[0] // Получаем первый байт строки (символ "Y")
fmt.Println(firstLetter) // Вывод: 89 (код символа "Y" в таблице Unicode)
```

<mark style="background: #FFF3A3A6;">Тип данных byte в Go — беззнаковый 8-битный целочисленный тип, может принимать значения от 0 до 255.</mark>

Руны — символы из набора Unicode в Go. Каждая руна — это один символ, который может быть представлен числовым значением типа `rune`.

Адаптируем наш предыдущий код под руны:

```go
name := "異體字"
firstLetter := []rune(name)[0] // Конвертируем строку в слайс рун
// И с помощью [0] забираем первый элемент
fmt.Println(firstLetter) // Вывод: 30064
fmt.Println(string(firstLetter)) // Вывод: 異
```

Литерал руны — это её константа, целочисленное значение, идентифицирующее кодовую точку Unicode. Простыми словами, это численное значение символа в Unicode. Литерал руны можно задать с помощью одинарных кавычек — например, `'x'`.

## Методы
`strings.Join` объединяет строки в одну строку с указанным разделителем:

```go
package main

import (
    "fmt"
    "strings"
)

func main() {
    words := []string{"Это", "несколько", "слов"}
    joined := strings.Join(words, " ")
    fmt.Println(joined) // Вывод: Это несколько слов
}
```
